# N1 光影相册 v2.0

一个专为诺基亚 N1 平板设计的相册互传工具，支持局域网内零配置自动发现，手机上传照片后平板自动裁剪为 4:3 比例。

## 功能

### 平板模式（Server）
- 启动相册服务器，局域网内广播自身
- 接收手机上传的照片
- 自动裁剪为 4:3 居中比例
- 网格展示所有照片
- 长按多选删除

### 手机模式（Client）
- 自动扫描局域网内的平板
- 零配置连接，无需输入 IP
- 浏览平板相册
- 批量上传照片
- 远程删除照片

### 图片裁剪
- 检测宽高比，4:3 ±0.03 容差内跳过
- JPEG 质量 95%，PNG 无损
- 自动修正照片方向（Exif）
- 居中裁剪

## 使用方式

### 平板端
1. 打开 APP，选择「平板模式」
2. 服务器自动启动，等待手机连接
3. 手机上传的照片会自动出现在相册中

### 手机端
1. 打开 APP，选择「手机模式」
2. 自动搜索局域网内的平板
3. 点击发现的设备连接
4. 点击右下角 + 上传照片
5. 长按照片可删除

## 构建 APK

使用 GitHub Actions 自动构建：

1. 把本项目代码推送到 GitHub 仓库
2. 进入 Actions 页面
3. 等待构建完成，下载 Artifacts 中的 APK

详细步骤见项目构建指南。

## 技术架构

- Kotlin + Android SDK 34
- Material Design 3 界面
- Ktor HTTP 服务器/客户端
- Android NSD 零配置发现
- Coil 图片加载
- kotlinx.serialization 序列化
